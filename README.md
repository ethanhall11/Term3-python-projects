# Term 3 — Python Projects

Python projects from my self-directed Term 3: Technology, Data & Systems.

This repo contains my work from CS50P Problem Set 0 (Harvard's Introduction to Programming with Python). All five problems were submitted to CS50's autograder and passed 4/4 tests each.

## Files

- **indoor.py** — Converts user input to lowercase
- **playback.py** — Replaces spaces with `...` for slow-mo speech effect
- **faces.py** — Replaces `:)` and `:(` with emoji 🙂🙁
- **einstein.py** — Calculates energy from mass using E = mc²
- **tip.py** — Tip calculator using custom functions for parsing dollar/percent inputs

## Built using

- Python 3
- CS50P (Harvard) Week 0 concepts: variables, functions, type conversion, string methods, f-strings

Part of a 16-week self-directed Term 3 covering Python, data literacy, AI fluency, and personal finance. Started April 2026.
